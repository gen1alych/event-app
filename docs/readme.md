# Event App
Mobile application for organizing events.

## Features
- Create events
- Manage participants

## Setup Instuctions
1. Clone the repository
2. Open index.html in browser
3. Start organizing events

### Project Structure
- **src/** → contains the main HTML files and source code  
- **docs/** → project documentation  
- **commits.txt** → exported commit history  
- **readme.md** → main project description

### Key Git Operations in VS Code
- **Initialize Repository**: Ctrl+Shift+G → Initialize Repository  
- **Commit**: Source Control → ➕ → ✔ Commit  
- **Push/Pull**: Source Control → ⋯ → Push/Pull  
- **Create Branch**: Click on the branch name at the bottom → Create new branch  
- **Merge**: Ctrl+Shift+P → Git: Merge Branch  

### Alternative path
Revert commit used to restore previous state when changes need to be undone.

Note: Asynchronous fetch for remote changes performed.
